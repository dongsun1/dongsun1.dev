## 프레임워크 및 라이브러리

- ### **Next.js**

  SSR을 통해 SEO에 유리하게 하기 위해 사용했습니다.

  블로그 만들기에 유용한 라이브러리가 많다고 판단하여 사용했습니다.

- ### **tailwind**

  빠른 개발과 일관성 있는 디자인을 위해 사용했습니다.

- ### **next-theme**

  라이트 모드와 다크 모드 기능 개발을 위해 사용했습니다.

- ### **react-markdown**

  마크다운을 HTML로 변환하기 위해 사용했습니다.

- ### **material-ui**
  빠른 개발을 위해 사용했습니다.
